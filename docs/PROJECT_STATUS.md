## Scientific bottom line — SCOPE CORRECTION (four-audit forensic synthesis `756ba1cd`; all claims verified against code/data)
**HEADLINE (four-audit forensic synthesis `756ba1cd`, `AUDIT_SYNTHESIS.md`): the project falsified a
PROXY of its hypothesis, not the hypothesis.** Every "DEAD" verdict rests on gene expression, a PCA of
expression, or a DNA/WES antigen-quantity proxy — never on the non-reference transcriptome features
(splicing, intron retention, TE/ERV, fusions, cryptic ORFs) the hypothesis is actually about. Honest
status of the central claim: **largely UNTESTED, not falsified.** What DOES stand (well-powered): RNA
antigen *quantity* (DNA/WES proxy) carries no ICB signal, and immune composition is a real cross-cohort
predictor. Forensic detail below (all verified against code/data, not summary docs):
- The kill-test "learned representation" arm (`analysis/kill_tests/headtohead_repr.py`) is
  literally `PCA(top-2000 variable genes)` — a linear summary of bulk expression, NOT a
  foundation-model reconstruction of evolutionary RNA state.
- The EVA "encoder" head-to-head collapses to expression PCA — **proven empirically** by the overview
  session's forensics (`ENCODER_EVALUATION_FORENSICS.md`, `eva_linearity_proof.png`, read here). The
  per-patient EVA feature is a fixed linear map of the 997 reference-gene expression submatrix:
  predicting all 1024 EVA dims from the row-normalized genes gives **R²=1.000000**; EVA feature PC1
  tracks expression PC1 at r=0.957. So the EVA-vs-PCA head-to-head was "one linear function of bulk
  expression vs another," guaranteed to tie — its LOCO 0.414 is a DESIGN ARTIFACT, not a verdict on EVA.
  No patient-specific/aberrant sequence was ever embedded (997 fixed reference cDNAs, 1.59% of features).
- **Orthrus was NEVER run at patient scale** (overview forensics, verified against its artifacts here):
  its only outputs are `orthrus_bench.json` (a 5-sequence CPU-throughput benchmark on housekeeping genes
  B2M/GAPDH/ACTB/CD8A/TP53) and a feasibility memo — no patient features, no cohort split. The "~0.49
  Orthrus LOCO" cited in some docs is a **misattributed expression-PCA baseline number**, not an Orthrus
  result. So the honest count is: **zero encoders were tested on patient-specific/aberrant sequence** —
  EVA collapses to expression by construction, Orthrus never ran. (Corrects the earlier "two-encoder null"
  framing in this doc.)

  intron-retention, fusions) — were NEVER built at cohort scale. Per the Orthrus session's own
