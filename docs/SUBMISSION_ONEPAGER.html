Does bulk tumor RNA encode a latent evolutionary state that predicts immunotherapy response?

evolutionary-rna-state · Built with Claude: Life Sciences hackathon · 2026-07-11

One line: We built the first cohort-scale non-reference RNA feature matrix and the learned-representation (Evo 2) arm the hypothesis calls for, then ran pre-registered two-block tests against an established immune-composition floor. The non-reference layer carries no independent, portable ICB-response signal — and we identify the mechanism why.

The hypothesis, tested on its own terms

The claim: a tumor’s bulk transcriptome contains an implicit evolutionary readout — alternative splicing, RNA editing, intron retention, transposable-element activity, novel junctions — that predicts checkpoint-blockade response beyond what gene expression already tells you. Every prior verdict in this project used expression, expression-PCA, or a DNA proxy. This is the first time the hypothesis’s own features were computed at cohort scale and tested.

What we built

  • Unified Apple-Silicon Nextflow workflow — one DSL2 spine fanning a BAM channel to the editing / intron-retention / TE-ERV / splicing subworkflows (-profile apple_silicon).
  • First cohort-scale non-reference matrix — RNA editing, IR, splicing burden, TE/ERV family activity across three melanoma ICB cohorts.
  • Learned-representation arm on Modal GPUs — Evo 2-7B scoring the sequence aberrancy of novel splice junctions (local Apple GPU is unreachable from the sandbox; all FM scoring ran on Modal H100/A100).

The result: a null — in the only valid evaluation frame

Cross-cohort (LOCO) evaluation is invalid for this 3-cohort set: the immune floor itself fails to transfer across cohorts (its response-direction flips), so no feature block is interpretable cross-cohort. The valid frame is within-cohort. We ran the Evo 2 two-block test in the best-powered valid frame (Gide) and replicated it in a second independent within-cohort test (Hugo).

The insight: why the whole non-reference layer comes up empty

The hypothesis’s real claim is not “aberrant junctions predict response,” but that a junction that is both sequence-surprising and abundantly expressed — the aberrant-and-expressed event — carries the signal. We built that aberrancy×dosage interaction explicitly and tested it. It is also null (residual p=0.92 Gide / 0.22 Hugo) — but informatively: it scores 0.612 alone in Gide, then collapses to 0.421 once the floor is removed. That is the fingerprint of a proxy.

Mechanism. The aberrant-expressed junction burden is a correlate of the inflamed-tumor state (Gide: GEP ρ=+0.36, IFNγ +0.29) — and its coupling to inflammation flips sign across cohorts (same-feature GEP: burden +0.36 Gide → −0.15 Hugo; aberrant-abundant fraction +0.31 → −0.37; systematic across all five inflammation axes). So the measurable non-reference “aberrancy” is entangled with immune composition and inherits the same cross-cohort direction-flip that sinks the immune floor itself. One unifying reason no non-reference layer — interpretable callers or an Evo 2 sequence model — ever adds independent, portable signal: they are downstream readouts of the tumor-immune state, seen through a lens that reorients between cohorts.

Can a model disentangle aberrancy from immune composition? — No

The obvious rescue: build a model that separates tumor-intrinsic aberrancy from the immune-composition component and recovers independent signal. A disentangling model is a model predicting response from the aberrancy component orthogonal to immune composition. We built that residual against the richest immune basis available — the 5 immune signatures plus all 11 InstaPrism deconvolution cell fractions (malignant/purity, CD8/CD4 T, Tregs, myeloid, B, plasma, pDC, mast, cycling), fold-contained.

The immune-orthogonal component is at/below chance in both cohorts. In Hugo the deconvolution correction pulls the residual down toward chance (0.576→0.488) — the cell fractions absorb the last apparent signal. This is not a modeling-power limit: a disentangling model can only recover a component that exists, and the orthogonal subspace is empty. It mirrors the project’s expression-level result (MEDIATION_PURITY_CORRECTED.md): after purity correction the phenotype→infiltration a-path collapses to zero (antigen-presentation +0.563 p=0.0002 → +0.047 p=0.64). Two matched tests — expression mediation and sequence disentangling — both land on composition-entanglement.

What would be needed to salvage the independent-signal claim

  • A feature that is aberrancy-driven but decoupled from bulk immune infiltration — e.g. per-neojunction HLA-presented peptide load (the presentation layer: patient HLA calling + MHC-binding on translated, NMD-escaping ORFs), not a bulk burden. Scoped in ENCODER_PHASE_PROTOCOL.md.
  • A regime where the immune floor transfers — a single large cohort with internal splits, not three small cohorts whose response-direction flips.
