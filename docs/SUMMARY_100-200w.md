# Project summary (171 words)

Does a single latent "evolutionary RNA-state" organize immune-checkpoint (ICB) response? On five public melanoma ICB cohorts we first reproduce the weak approved-biomarker ceiling (TMB, TIDE, clinical panel all AUROC 0.53-0.62). Testing the hypothesis on WES-derived neoantigen proxies is null: the RNA-phenotype proxies share no low-rank structure beyond mutational burden (permutation p=0.78) and add zero incremental AUROC over the floor. We argue this is diagnostic - exome annotations cannot see a transcriptomic state - and built a from-scratch, bandwidth-aware raw-read pipeline (salmon, arm64) to de-novo quantify 52 pre-treatment transcriptomes across three cohorts directly from ENA FASTQ. A de-novo antigen-presentation axis separates responders within one cohort (leave-one-out AUROC~0.87), but rigorous, leakage-free analysis shows it is largely infiltration-driven (Spearman 0.77 with leukocyte content) and fails to transfer to BOTH independent held-out cohorts (Riaz 0.36, Hugo 0.58). The honest conclusion: gene-level de-novo expression reflects the microenvironment and is cohort-specific; testing a transferable tumor-intrinsic RNA state requires the WES-blind, multi-phenotype features (intron retention, RNA editing, TE/ERV) that standard pipelines discard - infrastructure we built and staged.
