# analysis/baseline — genomic baselines

Standard-of-field genomic predictors the RNA-state model is benchmarked against.

## Modules
- `tmb_standardized.py` — standardized tumor mutational burden (TMB) computation.
- `snv_indel_neoantigen.py` — SNV/indel-derived neoantigen burden.
- `test_tmb_standardized.py` — unit validation for the standardized-TMB module.
- `test_snv_indel_neoantigen.py` — unit validation for the SNV/indel neoantigen module.
