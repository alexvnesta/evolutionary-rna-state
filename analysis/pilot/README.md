# analysis/pilot — de-novo Salmon pilot

The initial de-novo Salmon quantification pilot and its deepened, defensible follow-up.

## Modules
- `analyze_pilot.py` — turn the Salmon pilot quant into antigen-presentation / immune axes and a first response readout.
- `deepen_analysis.py` — scale the n=12 pilot to the full de-novo feature set, add tumor-intrinsic vs. infiltration decomposition and held-out external validation.
- `run_salmon_pilot.sh` — driver script to quantify the pilot sample set with Salmon.
